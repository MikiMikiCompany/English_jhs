1. 【魔法の呪文】現在完了進行形とは？

現在完了進行形は、「（過去のある時から）今までずっと〜し続けている」という状態を表す呪文だ。動作が今この瞬間も続いていることを強調したい時に使うんだぜ。

【基本の形】
have / has + been + 一般動詞のing形

ただの「現在進行形（〜している）」と「現在完了形（ずっと〜だ）」をミックスさせたような形だな。構造を分解するとこうなるぜ！

I have been practicing magic for two hours.
私は2時間ずっと魔法の練習をし続けています。
  • S（主語）： I
  • V（動詞）： have been practicing （「ずっと〜し続けている」という一つの塊だぜ！）
  • O（目的語）： magic
  • 修飾語： for two hours （「2時間の間」という期間を表すな）

2. 魔理沙先生の集中講義：ここがポイントだぜ！

💡

「ずっと」の魔法：
「今、勉強している」だけなら進行形（am studying）でいい。だが、「さっきから今まで、一瞬も休まずずっと勉強しているんだ！」と、その継続感を自慢したい（あるいは嘆きたい）ときは、この現在完了進行形の出番だぜ。

注意点：
状態を表す動詞（know, like, have [持っている] など）は、この魔法（ing形）には使えないのがルールだ。動作を表す動詞（play, read, wait など）と組み合わせるんだぜ。覚えておけよ！

3. 練習問題：スペルの調合に挑戦！

よし、学んだ知識を実際に使ってみるぞ。ケアレスミスという名の魔物に足をすくわれないよう、慎重に解いていこうぜ！

【問1：空所補充】
「彼女は午前中からずっと本を読んでいます。」という意味になるように、（　）に適する語を入れなさい。

She (       ) (       ) (       ) a book since this morning.
▶︎ 答えを見る
正解： has, been, reading
解説： 主語が She だから、have ではなく has を使うのが重要だぜ。その後ろに been + ing をつなげれば完成だ！

【問2：並び替え】
次の語を並び替えて、正しい英文を作りなさい。
[ long / waiting / you / have / how / been / ? ]

(                                              )
▶︎ 答えを見る
正解： How long have you been waiting?
和訳： あなたはどのくらい（の間）待っているのですか？
解説： 期間を尋ねる「How long」を文頭に置くんだ。疑問文だから have を主語の前に出すのを忘れるなよ！

【問3：英作文】
「昨日からずっと雨が降っています。」という文を英語で書きなさい。

(                                              )
▶︎ 答えを見る
正解：
It has been raining since yesterday.
解説： 天候を表す主語は It だな。It は3人称単数だから has を使うぜ。rain（雨が降る）を ing形にして、since（〜から）と組み合わせれば完璧なスペルの調合だ！
