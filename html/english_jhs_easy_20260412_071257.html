【魔理沙の魔法英文法教室】第1章：妄想を現実に！？仮定法過去の極意

「ようこそ、私の魔法教室へ！私は講師の魔理沙だぜ。今日は、現実にはありえない『もしも』の話をするための禁断の魔術、仮定法過去を伝授してやる。しっかりついてこいよ！」

★魔法の呪文（基本構文）

「仮定法過去は、今の現実とは違う『妄想』や『願い』を表現する時に使うんだ。ポイントは、現在の話なのに『過去形』を使うことだぜ。時間を一つ後ろにズラすことで、現実との距離感を出すんだな。」

【基本のカタチ】

If + 主語 + 動詞の過去形 ..., 主語 + 助動詞の過去形 (would / could) + 動詞の原形.

If I had a magic wand, I would fly in the sky.
（現実には持ってないけど）もし魔法の杖を持っていれば、空を飛ぶのになあ。

（[S] I / [V] had / [O] a magic wand）

■魔理沙先生の「スペル調合」ワンポイント！

「おい、ここが一番のひっかけポイントだ。仮定法過去で be動詞を使うときは、主語が I や he であっても、伝統的なルールでは were を使うのが基本だぜ。was でも間違いじゃない場合が増えてるが、入試という『儀式』では were を選ぶのが無難だ。覚えておけよ！」

If I were you, I would study harder.
もし私が君なら、もっと一生懸命勉強するのになあ。

（[S] I / [V] were / [C] you）← 現実にはありえないからこそ、特別な be動詞を使うんだぜ！

■練習問題：魔法の発動準備

「よし、理論は完璧だな。次は実際にスペル（英文）を組み立ててみよう。ケアレスミスは爆発の元だ、慎重に解けよ！」

【問1】次の（　）内の動詞を、仮定法過去の形に書き換えなさい。

If she ( know ) his phone number, she would call him.

答えと解説を見る

答え：knew

If she knew his phone number, she would call him.
もし彼女が彼の電話番号を知っていれば、彼に電話するだろうに。

【解説】現在の事実（番号を知らない）に反する仮定だから、動詞の know を過去形 knew に変えるんだぜ。後半に would call と助動詞の過去形があるのが、仮定法の合図だな！

【問2】日本文に合うように、次の語句を並べ替えなさい。

「もし時間がたっぷりあれば、私は世界中を旅行できるのに。」

( I / around / could / if / had / time / enough / world / travel / the / . )

答えと解説を見る

答え：If I had enough time, I could travel around the world.

もし私が十分な時間を持っていれば、世界中を旅行できるのに。

【解説】「～できるのに」という可能性を表すときは、助動詞 can の過去形 could を使うのがミソだぜ。If 節の中の have も過去形 had にするのを忘れるなよ！

【問3】次の「現実の文」を、「仮定法の文」に書き換えなさい。

As I am not a bird, I cannot fly to you.（私は鳥ではないので、君のところへ飛んでいけません。）

→ If I (          ) a bird, I (          ) fly to you.

答えと解説を見る

答え：were / could

If I were a bird, I could fly to you.
もし私が鳥なら、君のところへ飛んでいけるのに。

【解説】これは入試でも頻出の「書き換え魔法」だ！現実が否定文（～ない）なら仮定法は肯定文（～なら）に、現実が肯定文なら仮定法は否定文にするのが鉄則だぜ。be動詞はさっき言った通り were を使うのがスマートだな！
