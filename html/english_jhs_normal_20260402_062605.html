レベル：標準 (Normal)

【魔理沙の魔法英語教室】～助動詞の魔導書を解読するぜ！～

よう、よく来たな！今日は英文の雰囲気を一気に変える「触媒」、助動詞について伝授してやるぜ。助動詞は動詞の前に置くだけで、「～しなければならない」とか「～してもよい」といった魔法のニュアンスを付け加えることができるんだ。まさに「スペルの調合」には欠かせない素材だな！

1. 助動詞の基本ルール（魔法の制約）

助動詞を使うときは、絶対に守らなきゃいけないルールが2つあるぜ。これを破ると魔法が暴発するから気をつけな！

  • ルール①： 助動詞の後は必ず動詞の原形（元の形）を置くこと！sをつけたり過去形にしたりするのは厳禁だ。
  • ルール②： 助動詞は2つ並べて使えない（must willなどはダメだぜ）。

2. 入試頻出！「義務」と「不必要」の使い分け

ここが一番の「落とし穴」だ。しっかり見極めるんだぜ！

You must read this book.
あなたは（絶対に）この本を読まなければならない。

※ must は強い強制力を持つ魔法だ。話し手の強い意志がこもっているぜ。

You have to read this book.
あなたは（ルールや状況的に）この本を読む必要がある。

※ have to は「客観的な必要性」だ。入試では must の書き換えとしてよく出るが、否定形になると意味が激変するから要注意だぜ！

魔理沙先生：「おい生徒！ must not と don't have to の違いを言ってみろ！」

生徒：「えーっと、どっちも『～してはいけない』じゃないんですか？」

魔理沙先生：「甘いぜ！そんなんじゃ魔力切れで不合格だ！ must not は『禁止（～してはいけない）』だが、 don't have to は『不必要（～しなくてもいい）』という意味になるんだ。この違いは入試で狙われる最大のポイントだぜ！」

【修練の刻】練習問題に挑戦だぜ！

第1問（空所補充）： 次の日本文の意味になるように、（　）に適切な語を入れろ！

「あなたは明日、早く起きる必要はありません。」

You (      ) (      ) (      ) get up early tomorrow.
▶ 答えを見るぜ！

答え：don't have to

【解説】 「～する必要はない」は不必要を表す魔法 don't have to を使うんだ。 must not を使うと「起きてはいけない（寝てろ！）」という禁止の意味になっちまうからな。主語が3人称単数なら doesn't have to になるのも忘れるなよ！

第2問（並び替え）： 次の語句を並べ替えて、正しい呪文（英文）を完成させろ！

「私たちはこの部屋で写真を撮ってはいけません。」

( take / mustn't / photos / in / we / room / this / . )

▶ 答えを見るぜ！
We mustn't take photos in this room.

【解説】 「～してはいけない」という強い禁止は must not (mustn't) の出番だ。語順は 【主語(S) ＋ 助動詞 ＋ 動詞の原形(V) ＋ 目的語(O)】。この文型構造をしっかり守るのが、美しい魔法（英文）を作るコツだぜ！

第3問（英作文）： あなたの知識を総動員して、英語に調合（翻訳）してみろ！

「あなたは毎日、英語を練習すべきです。」（「～すべき」は should を使えよ！）

▶ 答えを見るぜ！
You should practice English every day.
（ S: You / 助: should / V: practice / O: English ）

【解説】 should は「～すべきだ」「～したほうがいい」というアドバイスの魔法だ。 practice は動詞の原形になっているな？ every day などの時を表す言葉は文の最後に添えるのが一般的だぜ。毎日コツコツやるのが、最強の魔導師への近道だな！

【魔理沙のまとめ】

助動詞は、話し手の気持ちを乗せる「感情のスパイス」だ！

  • must: 強い義務（～しろ！）、否定は「禁止」
  • have to: 客観的な義務、否定は「不必要」
  • should: 軽い義務・助言（～したほうがいいぜ）
  • may: 許可（～してもいいぞ）や推量

今日の講義はここまでだ！この「魔法の呪文」を使いこなして、入試という名のダンジョンを攻略してこいよ！応援してるぜ！
