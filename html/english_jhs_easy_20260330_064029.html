レベル：基本 (Easy)

霧雨魔法店・英語特別講義：二つの文を繋ぐ「魔法の接着剤」

魔理沙：よぉ！今日も元気に英語の修行に励んでいるな！私は霧雨魔理沙だぜ。
今日は、バラバラの呪文（文）をガッチャンコと一つに繋ぎ合わせる強力な魔法、「接続詞」を伝授してやるぜ！これを使えば、より長く、より強力な魔法（英文）を放つことができるようになるんだ。

霊夢：「接続詞」って、 when とか because とかのこと？なんだか種類が多くて、どこに置けばいいのか迷っちゃうのよね。

魔理沙：安心しろ！接続詞は「文と文を繋ぐ接着剤」だと考えればいい。基本の型はこれだぜ！
【接続詞 + 主語(S) + 動詞(V) , 主語(S) + 動詞(V)】、または
【主語(S) + 動詞(V) + 接続詞 + 主語(S) + 動詞(V)】だ！

1. 時と条件の魔法：when と if

魔理沙：まずは基本中の基本、「～のとき」を表す when と、「もし～なら」を表す if だ。

When I was a child, I liked magic.
私が子供だったとき、私は魔法が好きでした。
If it is sunny tomorrow, let's go to the forest.
もし明日晴れたら、森へ行きましょう。

霊夢：文の最初に when や if が来る時は、真ん中に カンマ（ , ） が必要なのね。

魔理沙：その通りだぜ！文の順番を入れ替えて I liked magic when I was a child. とすることもできるが、その場合はカンマは不要だ。呪文の構成要素の配置を間違えないように集中しろよ！

2. 理由と中身の魔法：because と that

魔理沙：次は、理由を説明する because と、「～ということ」を後ろに繋げる that だぜ！

I am happy because I found a rare mushroom.
私は珍しいキノコを見つけたので（から）、幸せです。
I think that English is very important.
私は英語がとても重要だ（ということ）と思います。

霊夢：that はよく省略されるって聞いたわよ？

魔理沙：おっ、鋭いな霊夢！ think（思う）や know（知っている）の後ろの that は、まるで透明化の魔法がかかったみたいに省略されることが多いんだ。だが、後ろに「主語＋動詞」が続く構造に変わりはないぜ！

【演習問題：スペルの調合に挑戦だ！】

魔理沙：さあ、ここまでの知識を使って練習問題を解いてみるぜ！凡ミスをしたら、私の八卦炉で少し炙ってやるから覚悟しろよな！

問1：次の空所に最も適切な接続詞を入れなさい。

(　　　) you have time, please help me with my magic practice.

（訳：もし時間があれば、私の魔法の練習を手伝ってください。）

▶ 答えを見る

正解：If

【解説】「もし～なら」という条件を表す魔法の呪文は If だぜ。文の先頭に来るから、大文字で始めるのを忘れるなよ！

問2：次の日本文の意味になるように、( )内の語を並び替えなさい。

私は、彼が最強の魔法使いだと信じています。
( believe / he / that / is / I / the strongest wizard ).

▶ 答えを見る
I believe that he is the strongest wizard.

【解説】「～だと信じている」という構造だ。 I believe（私は信じる）の後に、内容を繋ぐ接着剤の that を置き、その後に「彼が～だ」という文を繋げるんだぜ！

問3：次の日本文を英語に直しなさい（英作文）。

雨が降っていたので、私は家にいました。

▶ 答えを見る
I stayed home because it was raining.

（または Because it was raining, I stayed home.）

【解説】理由を表す because を使うんだ。過去のことだから、 stay を stayed に、 is を was にする時制の管理を忘れるな！これは呪文の「成分調整」と同じくらい大事なことだぜ！

霊夢：ふぅ、なんとか解けたわ。接続詞が二つの文の「橋渡し」をしているイメージね。

魔理沙：その意気だ！接続詞を使いこなせれば、君の英語は一気に表現力が増す。まさにスペルカードの同時発動のようなもんだ！しっかり復習して、次のレベルの魔法に備えておけよな。今日はここまで。お疲れさんだぜ！
