魔理沙先生の爆速・魔法文法：接続詞の巻

よぉ、よく来たな！私は魔理沙先生だぜ。今日は文法界の「接着剤」であり「連結魔法」、接続詞について教えてやるぜ。 バラバラの言葉を繋いで、一つの大きな魔法（文章）にするための重要なテクニックだ。これをマスターすれば、お前の英作文は「ただの単語の羅列」から「強力な魔導書」へと進化するぜ！

【魔法の呪文：接続詞の基本ルール】

接続詞には大きく分けて2つのタイプがあるんだぜ。

  1. that節（名詞のカタマリを作る魔法）: 「～ということ」という意味の巨大な名詞を作るぜ。
  2. 副詞節（時・条件・理由を添える魔法）: when（～のとき）、if（もし～なら）、because（なぜなら～なので）といった、状況を説明するパーツを作るぜ。

1. 思考を伝える that の魔法

「私は～だと思う」と言いたい時は、I think の後ろに that + 文 を繋げるんだ。この that は「ここから先は内容だぜ！」という合図だな。

S(I) V(believe) [O(that S'(English) V'(is) important)].
私は英語が重要であると信じています。

文全体の動詞(believe)に対して、that以下がまるごと目的語(O)になっているのがポイントだ。ちなみに、このthatはよく省略されるから、消えていても驚くんじゃないぜ。

2. 状況を操る魔法（when, if, because）

これらは文の先頭に置くこともできるし、文の途中に差し込むこともできる。先頭に置くときは、途中に コンマ(,) を打つのがルールだぜ！

When I am free, I read books.
暇なとき、私は本を読みます。
I will stay at home if it rains tomorrow.
もし明日雨が降れば、私は家にいます。

おっと、ここで重要な魔導知識だ。if や when の中では、未来のことでも現在形を使うっていう「時制の罠」がある。受験ではここを狙われるから、スペルの調合には気をつけな！

魔理沙先生の補足講義：

「先生！ because と so の違いは何だぜ？」ってよく聞かれるが、because は「原因」を強調し、so は「結果」へ繋ぐイメージだ。

Because it was cold, he wore a coat. (原因重視)
It was cold, so he wore a coat. (結果重視)

同じ意味でも、文の構造が違うんだ。一気に覚えると頭がオーバーヒートするから、まずは because の呪文を完璧にするんだぜ！

【演習問題：スペル調合テスト】

さぁ、ここまでの知識を使って、練習問題を解いてみるぜ。凡ミスは禁物だぞ！

第1問：空所補充

次の日本文の意味になるように、（　）に適する語を入れなぜ。

「私は彼女が親切な先生であることを知っています。」

I know (        ) she (        ) a kind teacher.
▶正解と解説を見る

正解： ( that ) / ( is )

解説： 「～ということ」を知っている、という構造にするために接続詞の that を使うぜ。後ろは「彼女＝先生」という文（S+V）が続くから、三人称単数現在の is を入れるのが正解だ。

第2問：並び替え

次の語句を並び替えて、正しい英文を完成させなぜ。（文頭に来る語も小文字にしてあるぜ）

「もしあなたが忙しいなら、私はあなたを助けましょう。」

( you / help / I / if / busy / will / are / , ).
▶正解と解説を見る

正解： If you are busy, I will help you.

解説： 「もし～なら」の部分を先に作るパターンだな。接続詞 if を文頭に置いた場合は、必ず途中に コンマ(,) を入れるのが鉄則だぜ。もちろん "I will help you if you are busy." でも正解だが、今回はコンマが含まれているから、この語順がベストだ！

第3問：英作文（スペルの調合）

次の日本文を英語に直しなぜ。

「彼はとてもお腹が空いていたので、そのケーキを全部食べました。」

※「～なので」には because を使いなぜ。また「全部食べた」は ate all of... を使いなぜ。

▶正解と解説を見る

正解例1： Because he was very hungry, he ate all of the cake.

正解例2： He ate all of the cake because he was very hungry.

解説： 原因（お腹が空いていた）を because で繋ぐぜ。時制に注目だ！「空いていた」「食べた」と過去の話をしているから、was や ate を使う必要があるぜ。過去形を使い忘れるのは、魔法の杖を忘れて戦場に行くようなもんだ。気をつけな！

魔理沙先生のラストアドバイス

入試では if や when の節の中で will を使っちゃうケアレスミスが多発するんだ。そこをしっかり見直すだけで、他の奴らに差をつけられるぜ。
