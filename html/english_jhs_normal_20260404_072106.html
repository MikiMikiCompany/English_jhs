レベル：標準 (Normal)

【魔法の継続術】現在完了進行形を極めるぜ！

よぉ、よく来たな！今日は「ずっと〜し続けている」という状態を表す、最高にエネルギッシュな魔術、現在完了進行形を伝授してやるぜ。この構文は、過去から今までアクションが途切れずに続いている様子を表すんだ。まさに「無限ループの魔法」ってところだな！

1. 基本の「スペルの調合」（構文）

この魔法を発動させるには、3つの材料を正しい順序で並べる必要があるんだ。一つでも間違えると、文法が暴発して減点対象になっちまうから気をつけろよ！

【基本形】 主語 + have / has + been + 動詞のing形 〜 .
意味：「（過去から今まで）ずっと〜し続けている」
He has been cleaning his room for two hours.
彼は2時間ずっと自分の部屋を掃除し続けているぜ。

解説：
S（主語）は He だから、助動詞は has を選ぶのが鉄則だ。been は固定のパーツで、その後に「掃除する」という動きを表す cleaning を繋げる。これで「過去に掃除を始めて、今もまだホウキを振り回している」という臨場感が出るんだな！

2. 魔理沙先生のワンポイント・レッスン

生徒：「先生！普通の『現在完了形（継続）』と何が違うんですか？ どっちも『ずっと〜している』って訳せますよね？」

魔理沙：「いい質問だぜ！そこが試験で狙われる『ひっかけの霧』だな。現在完了進行形は『動作』が今もバリバリ続いている時に使うんだ。逆に、『住んでいる(live)』とか『知っている(know)』みたいな『状態』を表す動詞は、進行形にできないのが英語のルールだ。だから状態の継続は普通の現在完了形で書くんだぜ！」

  • 〇 I have been playing tennis... （テニスは動作だから進行形OK！）
  • × I have been knowing her... （知っているのは状態だから、進行形にすると魔法が不発になるぜ！）

3. 実践演習：魔法のスペルを完成させろ！

ここからはお前の腕試しだ。入試レベルの問題を用意したぜ。凡ミスに気をつけて、完璧な英文を調合してくれ！

【問題1：空所補充】

（　）内の動詞を適切な形に変えて、次の日本文に合う英文を完成させろだぜ。
「彼女は午前中からずっとその本を読み続けています。」

She (      ) (      ) (      ) the book since this morning. (read)

【問題2：並び替え】

日本文の意味に合うように、語句を正しく並び替えろ。文頭に来る語は大文字にするんだぜ。
「あなたはどのくらいの間、ここで待っているのですか？」

( waiting / how long / been / you / have / here / ? )

【問題3：英作文（スペルの調合）】

次の日本文を英語に直せ。これができればお前も立派な魔術師だ！
「昨夜からずっと雪が降り続いています。」

4. 解答と解説だぜ！

【解答1】

She has been reading the book since this morning.
（彼女は午前中からずっとその本を読み続けています。）

解説：主語が She だから has を使い、現在完了進行形の公式「has + been + ing」を当てはめる。「〜からずっと」を表す since との相性もバッチリだな！

【解答2】

How long have you been waiting here?
（あなたはどのくらいの間、ここで待っているのですか？）

解説：期間を尋ねる How long を文頭に置くのがポイントだ。疑問文の形にするから、have を主語 you の前に出すのを忘れるなよ！

【解答3】

It has been snowing since last night.
（昨夜からずっと雪が降り続いています。）

解説：天候を表すときの主語は It を使うのが英語の決まりだ。snow（雪が降る）という動作が継続しているから、現在完了進行形が最適だぜ。もしここを計算ミス……じゃなかった、構文ミスして It is snowing にしちまうと、「今降っている」というだけの意味になっちまうから要注意だ！
