魔理沙先生の魔導書：現在完了進行形の極意

よう、よく来たな！今日は完了形の中でも、とりわけ「躍動感」あふれる強力な呪文、現在完了進行形を伝授してやるぜ。この構文は「過去から今までずっと、休まず動作を続けている」という状態を表す。普通の現在完了形よりも「ずっとやってるんだぜ！」という強調のニュアンスが強いのが特徴だ。しっかりマスターして、文法のスペルカードを使いこなそうぜ！

1. 魔法の構文（基本ルール）

現在完了進行形の基本形はこれだ！

have / has + been + V-ing（動詞のing形）

「（ずっと）～し続けている」という意味になる。現在完了形の「継続」用法と似ているが、現在完了進行形は「動作が今も絶え間なく続いている」ことに焦点を当てるんだぜ。

I (S) have been searching (V) for the magic stone (O) since this morning.
私は今朝から（ずっと）その魔石を探し続けている。

※ SVOC構造：Iが主語(S)、have been searchingが動詞句(V)、for the magic stoneが修飾語を伴う目的語的な役割(O相当)だな。

2. 魔理沙の特別講義：使い分けの罠

生徒：魔理沙先生！「ずっと知っている」なら have been knowing でいいんですよね？
魔理沙：待て待て！それは「禁忌のスペル」だぜ。knowやlike、have（持っている）みたいな状態動詞は、進行形にはできないんだ。継続を表したいなら、普通の現在完了形を使うのが鉄則だぜ。

3. 応用問題：スペルの調合（問題編）

さて、ここからは実戦だ。難関校の入試にも耐えうる、ひねりのある問題を３問用意したぜ。全問正解して、私の弟子を名乗ってみろ！

【第1問：書き換えの魔法】

次の2つの文がほぼ同じ意味になるように、空所に適切な語を入れろ。ただし、1語とは限らないぜ。

It started snowing three hours ago, and it is still snowing now.
It (     ) (     ) (     ) for three hours.

【第2問：並び替えの儀式】

日本文の意味になるように、( )内の語句を正しく並び替えろ。1語不要な語が含まれているから注意しろよ！

「アリスは1時間以上も、誰がそのクッキーを食べたのか問い詰め続けている。」

Alice ( been / has / asking / who / ate / the cookie / for / is / more / than ) an hour.

【第3問：究極のスペル調合（英作文）】

次の日本文を英語に直せ。現在完了進行形を必ず使うこと。

「霊夢は昨夜からずっと、新しい結界の術を練習するためにその庭に立っているのですか？」

4. 解答と魔理沙の解説

【第1問 解答】: has been snowing

解説：「3時間前に始まった」＋「今も続いている」の合わせ技だ。動作の継続を表すには現在完了進行形が最適だな。主語が It だから、have ではなく has を選ぶのを忘れるなよ。ケアレスミスは魔法の暴発の元だぜ！

3時間ずっと雪が降り続けている。

【第2問 解答】: Alice has been asking who ate the cookie for more than an hour.

不要な語：is

解説：「1時間以上」は for more than an hour と表現する。現在完了進行形（has been V-ing）の構造をしっかり作れば、あとは間接疑問文（who ate the cookie）を繋げるだけだ。「is」は現在進行形と混同させるための罠だぜ。惑わされなかったか？

【第3問 解答】: Has Reimu been standing in the garden since last night to practice a new barrier technique?

解説：疑問文だから、Has を文頭に持ってくるのが重要だぜ。SVOCで見ると、Reimu (S) / has been standing (V) / in the garden (修飾語) となる。「練習するために」は不定詞の副詞的用法 (to practice...) を使う。接続詞 since（～以来）との相性も抜群の構文だ。複雑な文だが、要素を一つずつ調合していけば完成するはずだぜ！
